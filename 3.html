고구려

1대 동명성왕 (東明聖王) / 기원전 37년 ~ 기원전 19년 / 주몽(朱蒙)
2대 유리왕(琉璃王) / 기원전 19년 ~ 서기 18년 / 유리(類利) / 1대 동명성왕의 장남(?)
3대 대무신왕(大武神王) / 18년 ~ 44년 / 무휼(無恤) / 2대 유리왕의 3남
4대 민중왕(閔中王) / 44년 ~ 48년 / 색주(色朱) / 2대 유리왕의 5남
5대 모본왕(慕本王) / 재위 48년 ~ 53년 / 해우(解憂) / 3대 대무신왕의 차남
6대 태조왕(太祖王) / 재위 53년 ~ 146년 / 궁(宮) / 2대 유리왕의 6남 재사의 장남
7대 차대왕(次大王) / 재위 146년 ~ 165년 / 수성(遂成) / / 2대 유리왕의 6남 재사의 차남
8대 신대왕(新大王) / 재위 165년 ~ 179년 / 백고(伯固) / / 2대 유리왕의 6남 재사의 3남
9대 고국천왕(故國川王) / 재위 179년 ~ 197년 / 남무(南武) / 8대 신대왕의 차남
10대 산상왕(山上王) / 재위 197년 ~ 227년 / 연우(延優) / 8대 신대왕의 4남
11대 동천왕(東川王) / 재위 227년 ~ 248년 / 우위거(憂位居) / 10대 산상왕의 장남
12대 중천왕(中川王) (재위 248년 ~ 270년) 동천왕의子 / 연불(然弗).
13대 서천왕(西川王) (재위 270년 ~ 292년) 중천왕의차남 / 약로(藥盧).
14대 봉상왕(烽上王) (재위 292년 ~ 300년) 서천왕의子 / 상부(相夫)
15대 미천왕(美川王) (재위 300년 ~ 331년) 서천왕의子인 돌고의 子 이름 을불(乙弗)
16대 고국원왕(故國原王) (재위 331년 ~ 371년) 미천왕의子 이름 사유(斯由)
17대 소수림왕(小獸林王) (재위 371년 ~ 384년) 고국원왕의子 이름 구부(丘夫).
18대 고국양왕(故國壤王) (재위 384년 ~ 391년) 소수림왕의弟 이름은 이련(伊連).
19대 광개토왕(廣開土王) (재위 391년 ~ 413년) 고국양왕의子 이름 담덕(談德)
20대 장수왕(長壽王) (재위 413년 ~ 491년) 광개토왕의子 이름 거련(巨連)
21대 문자명왕(文咨明王) (재위 492년 ~ 519년) 장수왕의 孫子 이름 나운(羅雲).
22대 안장왕(安臧王) (재위 519년 ~ 531년) 문자명왕의子 이름 흥안(興安).
23대 안원왕(安原王) (재위 531년 ~ 545년) 안장왕의弟 이름 보연(寶延).
24대 양원왕(陽原王) (재위 545년 ~ 559년) 안원왕子 이름 평성(平成).
25대 평원왕(平原王) (재위 559년 ~ 590년) 양원왕의子 이름 양성(陽成).
26대 영양왕(嬰陽王) (재위 590년 ~ 618년) 평원왕의子 이름 원(元)
27대 영류왕(榮留王) (재위 618년 ~ 642년) 평원왕의 차남이자 영양왕의 이복弟 이름 건무(建武)
28대 보장왕(寶臧王) (재위 642년 ~ 668년) 영류왕의弟 이름 장(臧)
